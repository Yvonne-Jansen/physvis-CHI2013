An eclipse project file is included in this repository. 

The main file for the experiment is called "ExperimentLauncher.java". It contains the class used for the first experiment, simply called "Experiment". In its current state, the second experiment is launched calling the class "ExperimentV2".




To use the prop with bluetooth on Mac OS:

1) type the following in a terminal:
sudo mkdir /var/lock
sudo chmod a+x /var/lock

2) setup bluetooth device named "Free2Move WU"
